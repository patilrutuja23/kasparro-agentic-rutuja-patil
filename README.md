# Multi-Agent Content Generation System

This repository contains a modular, agent-based automation system that converts structured product data into machine-readable content pages.

## Objective
Design a production-style multi-agent system that:
- Parses product data
- Generates categorized user questions
- Applies reusable content logic blocks
- Uses templates to assemble structured pages
- Outputs clean JSON files

## How to Run

```bash
python main.py
